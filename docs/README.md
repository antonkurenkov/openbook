# Openbook landing — mobile fixed

Это готовая замена содержимого папки `docs/` для GitHub Pages-лендинга `openbook`.

## Что исправлено

- Убран горизонтальный overflow на мобильных экранах.
- Header перестроен для телефонов: бренд + RU/EN сверху, навигация ниже в компактной сетке 4×1.
- Заголовки и карточки больше не вылезают за viewport.
- Hero, контейнер, финальный CTA и сетки обложек корректно складываются в одну колонку.
- Добавлены `scroll-margin-top`, чтобы sticky header не перекрывал якорные секции.
- CTA получили аккуратный персиково-веточный акцент вместо “дешёвых” стрелок.
- Обложка «Зелёной двери» подключена локально, без внешнего hotlink на raw.githubusercontent.com.

## Как применить

1. Распаковать архив.
2. Скопировать содержимое папки `openbook-landing-mobile-fixed/` в `openbook/docs/` с заменой файлов.
3. Закоммитить и запушить в `main`.
4. GitHub Pages обновит `https://antonkurenkov.github.io/openbook/`.

## Быстрый локальный просмотр

```bash
cd docs
python -m http.server 8000
```

Затем открыть `http://localhost:8000`.
