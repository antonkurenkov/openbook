# Зелёная дверь — GitHub Pages site

Лаконичный статический сайт для цикла «Зелёная дверь» в стиле high-key lineart.

## Состав

- `index.html` — страница лендинга.
- `styles.css` — цветовая система, сетки и адаптивность.
- `main.js` — reveal-анимация, активная навигация и просмотр изображений.
- `assets/` — favicon, OG-изображение и оптимизированные изображения обложек.
- `.nojekyll` — отключение Jekyll-обработки на GitHub Pages.

## Цвет

Основной зелёный взят с референса зелёной двери: `#437641`.
Тёмный вариант для текста и кнопок: `#244f2a`.

## Деплой

1. Распаковать архив.
2. Скопировать содержимое в папку `docs` репозитория `https://github.com/antonkurenkov/openbook`.
3. В настройках GitHub Pages выбрать публикацию из ветки `main`, папка `/docs`.

Сайт не требует сборки, зависимостей или внешних CDN.
