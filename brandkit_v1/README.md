# Зелёная Дверь — identity brand kit

Набор модульных SVG-элементов для сайта литературного проекта **«Зелёная Дверь»**.

## Состав

- `svg/00-brand-sheet.svg` — общий лист стиля: палитра, типографическое настроение, рамка, иконки.
- `svg/01-hero-threshold.svg` — hero-композиция с дверью, геометрией и тихими маршрутными линиями.
- `svg/02-hero-branching-container.svg` — абстрактная схема «ветвящегося контейнера» без акцента на дереве.
- `svg/emblem-zd.svg` — эмблема/монограмма.
- `svg/icons/*.svg` — набор минимальных иконок.
- `svg/components/*.svg` — рамки, разделители, фоновые паттерны, кнопочный акцент.
- `tokens/green-door-tokens.css` — CSS-переменные для сайта.
- `tokens/green-door-tokens.json` — палитра и правила в JSON.
- `prompts/*.txt` — исходный промпт и negative prompt.
- `preview/index.html` — локальная HTML-превью-страница.

## Палитра

- warm paper: `#FBFAF6`
- graphite: `#222522`
- pale line: `#D8D8CF`
- door green: `#427842`
- moss dark: `#2E5634`
- brass: `#B79A5B`

## Принцип

Стиль строится не вокруг дерева, а вокруг **двери как порога**, **архивной рамки**, **страницы**, **пути**, **узлов** и **ветвящейся структуры открытого издания**. Органика допускается только как тонкая абстрактная линия маршрута или едва заметная ботаническая реминисценция.

## Рекомендации для сайта

1. Использовать SVG как inline-элементы или background-image.
2. Hero не перегружать: много воздуха, один зеленый акцент.
3. Разделители и рамки применять как «архивную навигацию», а не как декоративный орнамент ради орнамента.
4. В текстах использовать широкую разрядку для заголовков и спокойный serif mood.
5. Не вводить дополнительные яркие цвета: зеленая дверь должна оставаться единственным сильным цветовым событием.
