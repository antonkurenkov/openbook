# Usage note

Все элементы собраны как редактируемые SVG без внешних шрифтов и без вложенных растровых изображений. SVG можно открывать в Illustrator, Figma, Inkscape или встраивать прямо в сайт.

Для production-версии сайта лучше:
- заменить SVG text на outline, если нужен полный контроль типографики;
- минифицировать SVG;
- вынести повторяющиеся цвета в CSS variables;
- адаптировать hero под реальные breakpoints.
